Operating Systems Terminolohy
Term Definition
Operating System The software that controls the overall operation of a computer.
Job Queue A job queue is a storage organization in which objects are ordered in FIFO fashion.
FIFO FIFO stands for first-in first-out, and means that objects are removed in the order in which they arrived.
Interactive processing The ability of an operating system that allowed a program being executed to carry dialogue, or communicate with the user.
Time-sharing An operating system that provided sevice to multiple users at the same time.
Multiprogramming Dividing time into short segments, each called a time slice, and then switching the CPU's attention among the processes, as each is allowed to execut for one time slice.
Application software Consists of the programs for performing task particular to the machinen's utilization, e.i. spreadsheets, program development software, and games.
System software Performs tasks that are common to computer systems in general.
Utility software Programs for performing activites that are fundamental to computer installations but not included in the operating system.
Graphical user interface (GUI) A modern user interface in which objects to be manipulated are represented pictorially on a display as icons.
Shell An older user interface that communicated to users through textual messages.
Kernel The internal part of an operating system. It contains software components that perform very basic functions required by the computer installation.
Device driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File manager A software component whose job is to coordinate the use of the machine's mass storage facilities.
Memory manager A component of an operating system's kernal which coordinates the machine's use of main memory.
Virtual memory The illusion of memory created by paging.
Boot straapping A procedure that transfers the operating system from mass storage inot main memory when the machine is turned on.
ROM Read-only memory, memory that can only be altered under special circumstances.
Deadlock The condition in which two or more processes are blocked from progressing becasue each is waiting for a resource that is allocated to another.
Spooling A deadlock avoidance scheme of holding data for output at a later, more convenient time.
Super user (or administration) A user that has highly privileged access to the operating system This user can alter settings within the operating system, modify critcal software packages, adjust the privileges granted to other users, and perform a variety of ther activties that are denied normal use.
Multi-factor authentication A second authentication factor other than a stardard password (a unique question, phone number, or fingerprint).
