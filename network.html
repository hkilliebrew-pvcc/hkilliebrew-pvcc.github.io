Networking & The Internet, Chapter 4 Terms:
Term Definition
Network Classifications LAN (Local Area Network)
  • A network consisting of a collecion of computers in a single building or building complex.
MAN (Mertopolitan Area Network)
  • A network of intermediate size, such as one spanning a local community.
WAN (Wide Area Network)
  • A network that links machines over a greater distance in neighboring cities or on opposite sides of the world.
Network models Bus network
  • A way/pattern in which machines are connected where machines are all connected to a common communication line.
Star network
  • When one machine serves as a central focal point to which all the others are connected.
Protocols CSMA/CD (Carrier Sense, Multiple Access with Collsion Detection) Ethernet
  • A bus network protocal that dictates the right to transmit messages.
CSMA/CA (Carrier Sense, Multiple Access with Collsiion Avoidance) Wifi
  • A star network protocal that priorizes avoiding collisions/network interference instead of detecting them.
Combining networks Repeater
  • A device that passes signals back and forth between two buses (usually with some form of amplification) without considering the meaning of the signals.
Bridge
  • Like a repeater, a bridge connects two buses; however, is does not pass all messaages across the connection. Instead, it forwards a message across the connection only when the message desination is on the other side.
Switch
  • Essentially a bridge with multiple connections, allowing it to connect several buses rather then just two.
Router
  • Special purpose computers used for forwarding messages. Note that the task of a router is different from that of repeaters, bridges, and switches in that routers provide links between networks while allowing each network to maintain its unique internal characteristics.
Gateway
  • The "point" at which one network is linked to an internet.
Client/server model P2P: Peer-to-peer
  • A communication model that provides service to and receives service from each other, rather than one server providing service to numerous others.
Distributed system
  • Software units that execute as processes on different computers.
Cloud computing
  • A huge pools of shared computers on a network can be allocated for use by clients as needed.
Internet Addressing Tier 1
  • Consist of a few very high-speed, high-capacity, international WANs. These networks are thought of as the backbone of the Internet.
Tier 2
  • Similar to Tier 1, but tend to be more reginal in scope and less potent in their capabilites.
Tier 3
  • An independent Internet operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP address ICANN ( Icann.org )
  • Internet Corporation for Assigned Names and Numbers is a non-profit corporation established to coordinate the Internet's operation.
DNS
  • Domain Name System is an internet-wide directory system used to translate domain names to IP addresses.
TLD: Top-level domain
  • Domains with suffixes following a period that are used to reflext the domain's classification. These include .com or commercial institutions, .edu for educational institutions, .gov ofr U.S gorvernment instututions, .org for nonprogit organizations, and .mil for military.
Country-code TLDs ( WorldStandards.eu )
  • Two-letter TLDs for each specific country, such as .au for Australia and .ca for Canada.
Internet protocols FTP
  • File Transfer Protocol An application for listing and copying files across a network.
HTTP
  • Hyper Text Transfer Protocol A protocol used to transfer documents between browers and webservers.
SMTP
  • Simple Mail Transfer Protocol Defines a way that two computers on a network may interact when transmitting an email message from one host to the other.
World Wide Web URL
  • Uniform resource locator A unique address used to locate and retrieve documents on the Web.
Client-side activities
  • Additional activity preformed by the client/browser.
Server-side activities
  • Additional activity preformed by the server/webserver.
Internet four-layer software Application layer
  • This layer prepares messages and provides destination addresses.
Transport layer
  • This layer accepts messages from the application layer and ensures that the messages are properly formated. This layer also divides long messages into smaller segments, known as packets, which can be transmitted over the Internet one segment at a time.
Network layer
  • This layer's job is to decide in which direction a packet should be sent at each step along the packet's path through the Internet.
Link layer
  • This layer has the responsiblility of transferring the packet. Thus, the link layer must deal with the communication details particular to the individual network in which the computer resides.
Internet is: Packet-switching network
  • A network which data is sent by breaking the data into blocks of bits called packets, these contain both the data being transmitted and information for routing the data.
Fault tolerant
  • The continued operation of a network when one or more of its components fail.
Redundant
  • There are more than one way to route data.
Cybersecurity Malware
  • Refers to malicious software.
Virus
  • Software tha infects a computer by inserting itself into programs that already reside in the machine.
Worm
  • An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
Trojan horse
  • A program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willlingly imported by the victim.
Spyware
  • Sometimes called sniffing software, is software that collects information about activities at the computer it resides on and repoirts that information back to the instigator of the attack.
Phishing
  • A technique of obtaining information explicitly by asking for it, typically carried out via email posing as a finacial institution, government bureau, or law enforcement agency.
DDoS attack
  • Distributed Denial of Service attack A Denial of Service attack requires the generation of a large number of messges in a short period of time in order to overwhelm the target server. When the work of generating the attacks is distrubued over many host, this is known as a distrubuted denial of service (DDoS).
Protection Firewall
  • A prevention program used to filter traffic passing thorugh a point in a network.
Spam filter
  • A variation of a firewall designed to block unwanted emails.
Proxy server
  • Software that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
Antivirus software
  • Software used to detect and remove the presence of known viruses and other infections.
Cryptography
  • The study of sending and receiving messages securely in the presence of adversaries.
Public key encryption
  • Techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
